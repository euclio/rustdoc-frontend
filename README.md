# rustdoc-frontend

A frontend that can be used with the new [rustdoc] to generate static
documentation.

## Usage

```bash
$ cargo run --release -- path/to/data.json
```

Documentation will be generated in `target/doc2`.

[rustdoc]: https://github.com/steveklabnik/rustdoc
